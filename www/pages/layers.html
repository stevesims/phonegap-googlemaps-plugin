Layers

You can overlay your map data.

  • GroundOverlay
  • TileOverlay

A rectangular image overlay on the map.

JavaScript

var bounds = [ new plugin.google.maps.LatLng(40.712216,-74.22655), new plugin.google.maps.LatLng(40.773941,-74.12544) ]; map.addGroundOverlay({ 'url': "http://www.lib.utexas.edu/maps/historical/newark_nj_1922.jpg", 'bounds': bounds, 'opacity': 0.5 }, function(groundOverlay) { map.moveCamera({ 'target': bounds }); });

map.addTileOverlay({ // <x>,<y> and <zoom> are replaced with actual values. tileUrlFormat: "http://tile.stamen.com/watercolor/<zoom>/<x>/<y>.jpg" }, function(tileOverlay) { });
